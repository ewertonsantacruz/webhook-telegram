Documentation

Installation

Follow these steps to install the Telegram Webhook Forwarder:

1. Prerequisites

  • Web server with PHP 7.4+ and MySQL
  • SSL certificate (required for Telegram webhooks)
  • Telegram Bot Token

2. Download and Setup


# Clone or download the files to your web directory
git clone https://github.com/yourusername/telegram-webhook-forwarder.git
cd telegram-webhook-forwarder

# Set permissions
chmod 755 .
chmod 755 public src config
chmod 644 public/*.php src/*.php
chmod 644 .htaccess

3. Database Setup


CREATE DATABASE telegram_forwarder;
USE telegram_forwarder;

CREATE TABLE telegram_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    chat_id VARCHAR(50),
    message TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Configuration

1. Update config.php

Edit config/config.php with your settings:


return [
    'telegram_bot_token' => 'YOUR_BOT_TOKEN',
    'webhook_url' => 'YOUR_DESTINATION_WEBHOOK_URL',
    'database' => [
        'host' => 'localhost',
        'username' => 'your_db_username',
        'password' => 'your_db_password',
        'database' => 'telegram_forwarder'
    ]
];

2. Set Telegram Webhook

Register your webhook URL with Telegram:


curl -F "url=https://your-domain.com/webhook-telegram/src/telegram_webhook.php" \
     https://api.telegram.org/botYOUR_BOT_TOKEN/setWebhook

Usage

Monitoring Messages

  1. Access the dashboard at https://your-domain.com/webhook-telegram/public/index.html
  2. Messages will appear automatically within 30 seconds
  3. Use the "Refresh Now" button for immediate updates

Testing the Webhook

Use the test endpoint to verify your setup:


curl https://your-domain.com/webhook-telegram/public/test_webhook.php

Troubleshooting

Common Issues

  1. Verify webhook is set correctly: curl https://api.telegram.org/botYOUR_BOT_TOKEN/getWebhookInfo
  2. Check database connection in config.php
  3. Review logs in logs/messages.log

Run these commands to fix permissions:


chmod 755 .
chmod 755 public src config
chmod 644 public/*.php src/*.php
chmod 666 logs/messages.log
