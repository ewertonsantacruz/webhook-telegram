Telegram Webhook Documentation

Endpoints

  • /index.php - Main webhook endpoint
  • /test_webhook.php - Test the webhook functionality
  • /test_db.php - Test database connection

Testing

To test the webhook:

  1. Configure your config.php with proper credentials
  2. Run test_db.php to verify database connection
  3. Run test_webhook.php to simulate a Telegram update

Logs

Logs are stored in:

  • File: /logs/messages.log
  • Database: telegram_logs table
